# 📊 Customer Churn Analysis Dashboard – Power BI

## 📌 Project Overview
This project focuses on analyzing **customer churn behavior** in a telecom company using **Power BI**.  
The goal is to understand why customers leave and identify the key factors influencing churn.

An **interactive dashboard** was developed to monitor churn trends and help businesses improve **customer retention strategies** through data-driven insights.

---

## 📊 Dataset

**Source:** Telco Customer Churn Dataset (Kaggle)

### Dataset Includes:
- Customer demographics
- Account information
- Services subscribed
- Billing details
- Churn status

---

## 🎯 Objective

- Measure the **customer churn rate**
- Identify **key factors influencing churn**
- Segment **high-risk customers**
- Provide insights to improve **customer retention strategies**

---

## 🔧 Tools & Technologies Used

- **Power BI**
- **Microsoft Excel** (Initial Data Review)
- **GitHub**

---

## 📈 Key Metrics Tracked

The following key performance indicators (KPIs) were analyzed:

- **Total Customers**
- **Churned Customers**
- **Churn Rate (%)**
- **Average Monthly Charges**
- **Average Customer Tenure**

---

## 📊 Dashboard Features

The Power BI dashboard includes multiple visualizations to understand churn patterns:

- **KPI Cards** showing overall churn performance
- **Churn Analysis by Contract Type**
- **Tenure-based Churn Trends**
- **Monthly Charges Impact on Churn**
- **Customer Segmentation** by:
  - Gender
  - Senior Citizen Status
  - Payment Method
- **Interactive Filters** for deeper analysis

---

## 🔍 Key Insights

From the analysis, several important insights were identified:

- Customers with **month-to-month contracts** have the highest churn rate.
- Customers with **short tenure** are more likely to churn.
- **Higher monthly charges** are associated with increased churn.
- Customers using **automatic payment methods** tend to churn less.
